    참고. 세부 메모¶

    아래 페이지는 통합 전 상세 메모를 참고용으로 남겨둔 것입니다. 메인 실습 루트는 Stage 0~4만 따라가면 됩니다.

    사람 확인 세부 메모¶

    사람이 직접 볼 때 특히 체크할 것¶

    • 모집인원 숫자 오독
    • 누락된 학과나 전형
    • 셀 병합 해석 실패
    • 날짜 형식 불일치
    • 표가 다음 페이지로 넘어가며 잘린 케이스

    사람 확인이 강한 이유¶

    • AI가 패턴을 잘 찾아도 업무상 중요한 미묘한 차이는 사람이 더 잘 잡습니다
    • 특히 "비슷하지만 다른 값"을 구분하는 데 강합니다
    • 마지막 확인은 품질 점검이자 다음 자동화 규칙 후보를 고르는 단계입니다

    정답 비교 세부 메모¶

    정답 비교에서 보는 것¶

    • 전체 정답률
    • 컬럼별 정답률
    • 틀린 칸 목록
    • 빠진 줄 목록
    • 추가로 생긴 줄 목록

    정답률 해석 팁¶

    • 점수 자체보다 반복 오류 패턴이 더 중요합니다
    • 특정 컬럼만 약하면 컬럼 매핑 규칙을 다시 봐야 합니다
    • 누락이 많으면 페이지 탐색이나 표 병합 로직을 의심해야 합니다
    • 오탐이 많으면 행 분리 기준이나 헤더 처리 규칙을 봐야 합니다
